# Monstera Landing Page

A responsive landing page for a plant-themed brand called **Monstera**, built using **HTML** and **Tailwind CSS**.  
This project features a clean hero section, navigation bar, CTA button, and plant care information cards.

---

## 🚀 Features

- Responsive Navbar
- Hero Section with Product Branding
- Call-to-Action Button
- Plant Care Info Section
- Tailwind CSS Utility Styling
- Google Fonts Integration
- Font Awesome Icons

## 📦 Installation

1. Download or clone the repository:

```bash
git clone https://github.com/your-username/monstera-landing-page.git
```

2. Open project folder:

```bash
cd monstera-landing-page
```

3. Run the project:

Simply open `index.html` in your browser.
